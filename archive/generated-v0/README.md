# Generated V0 Archive

## Purpose

This directory preserves the original generated SAVERSSYSTEM™ machine-readable records created during the architecture development process.

These files are retained for historical reference, traceability, and research.

## Status

The records in this directory are **not** the canonical machine entry points.

The canonical repository begins with:

- `index.json`

Supporting canonical records include:

- `identity.json`
- `architecture.json`
- `relationships.json`
- `entity-map.json`
- `metadata.json`
- `provenance.json`
- `version.json`

## Why this archive exists

During development, many model, index, status, and completion records were generated to explore and validate the SAVERSSYSTEM™ architecture.

Rather than delete those records, they are preserved here as part of the project's development history.

## Authority

Only the canonical records in the repository root should be treated as authoritative for machine discovery and interpretation.

Archived records remain available for reference but are not the primary source of truth.
